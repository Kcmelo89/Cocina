Sincronización — despensa actual: —

Cada despensa se sincroniza con su propio Bin en jsonbin.io.

Para crear un Bin nuevo:
1. Regístrate gratis en jsonbin.io
2. Clic en Create Bin → pega: {"frescos":[],"basicos":[],"especias":[]}
3. Copia el BIN ID y tu Master Key (Account → API Keys)
4. Pégalos aquí y pulsa Guardar

Para que otra persona use esta despensa, compártele el BIN ID y la Master Key. Ella crea la despensa con el mismo nombre y pega estos datos aquí.
